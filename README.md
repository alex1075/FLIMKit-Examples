# FLIMKit examples

This repoisitory contains example code for using FLIMKit, a software package for fluorescence lifetime imaging microscopy (FLIM) data analysis. The examples demonstrate how to use FLIMKit to analyze FLIM data, including loading data, performing lifetime fitting, and visualizing results.

## Getting Started

First download the FLIMKit software and install it on your computer. Then, clone this repository to access the example code.

```bash
git clone git@github.com:alex1075/FLIMKit-Examples.git
```

Make sure you install the FLIMkit Python package and its dependencies and set up your Python environment to use Jupyter notebooks. 

Copy the notebooks to the FLIMkit directory.

## WARNING
Do not commit the notebooks to the FLIMKit repository, as they may contain data and results that are not relevant to the main codebase. Instead, keep them in a separate directory or repository for your own use.
